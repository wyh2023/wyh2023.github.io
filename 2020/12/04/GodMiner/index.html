GodMiner

题目

Nezzar is addicted to game Gold Miner recently.

Gold Miner is a game in which the player is in an 𝑛×𝑛n×n grid, and the goal is to collect gold without entering forbidden area. It is known that cell (𝑖,𝑗)(i,j) is forbidden to enter if and only if $i=j$ or $i=j+1$.

In one round, Nezzar may choose two non-empty sets $X,Y⊆[n]$, and collect gold in cell $(i,j)$ for all $i∈X,j∈Y$. However, he immediately fails if there exists $i∈X$ and $j∈Y$ such that $(i,j)$ is forbidden to enter.

As a greedy miner, Nezzar would like to enter all but forbidden cells to collect gold. However, he is running out of time! He can only play the game for at most $50$ rounds. Can you design an algorithm to solve this problem for him?

Input

One line containing one integer $n (2≤n≤3000)$, the size of the grid.

Output

Please read this part carefully, as you may fail to pass the test cases if you have formating issues.

For each round with subsets $X,Y$ being chosen, output elements in $X$ increasingly in the first line, and then output elements in $Y$ increasingly in the second line.

Sample Input

3

Sample Output

1
2 3
1 2
3
3
1

Note

It is allowed to enter some cells multiple times!

Hint

Believe it or not, we can use divide-and-conquer technique to solve this task.

Let $T(n)$ denote the number of rounds we need to solve the above task with parameter $n$. Notice that $T(n)=T(n/2)+O(1)$ implies $T(n)=O(log⁡n)$, and we may need an algorithm satisfying this recurrence to solve the problem.

大致题意

有一个$n×n$的棋盘，在主对角线的两排是不允许开采的禁区，如下图。每一次我们可以选两个横纵坐标的集合$X$和$Y$。对于两个集合所有元素组成的点$(x, y)$进行开采。注意，这个点不可以是禁区。可以重复开采同一个点，但是总选择次数不可以超过50次（最多选100个集合）。任何有效的开采方法都会被判对。

Naive Solution

刚开始看这道题，以为这道题非常简单。因为感觉一眼就可以看出分治为左上角和右下角两个部分。然后写出了以下这样的代码：

  • 对于只有一个可开采的区域，直接返回这个开采的结果（下面的代码没写出来。。。）
  • 对于两个的情况，开采右上角；
  • 然后分奇偶情况，做不同的分割；
#include "iostream"
#include <cstdio>
using namespace std;

int n;
int cnt = 0;

void miner(int s, int left, int right, int up, int down){
    if(s == 1) return;
    if(s == 2) {
        printf("%d\n", up);
        printf("%d\n", right);
        printf("cnt=%d\n", cnt++);
        return;
    }
    int tmp = (s+1)/2;
    if(s%2 != 0){
        miner(tmp, left, left+tmp-1, up, down-tmp+1);
        miner(tmp, right-tmp+1, right, up+tmp-1, down);
        //右上
        for(int i=0; i<s-tmp; i++){
            printf("%d ", up+i);
        }
        printf("\n");
        for(int i=0; i<s-tmp; i++){
            printf("%d ", right-tmp+2+i);
        }
        printf("\n");
        printf("cnt=%d\n", cnt++);
        //左下
        for(int i=0; i<s-tmp; i++){
            printf("%d ", up+tmp+i);
        }
        printf("\n");
        for(int i=0; i<s-tmp; i++){
            printf("%d ", left+i);
        }
        printf("\n");
        printf("cnt=%d\n", cnt++);
    }
    if(s%2 == 0){
        miner(tmp+1, left, left+tmp, up, up+tmp);
        miner(tmp, right-tmp+1, right, down-tmp+1, down);
        //右上
        for(int i=0; i<tmp; i++){
            printf("%d ", up+i);
        }
        printf("\n");
        for(int i=0; i<tmp-1; i++){
            printf("%d ", left+tmp+1+i);
        }
        printf("\n");
        printf("cnt=%d\n", cnt++);
        //左下
        for(int i=0; i<tmp-1; i++){
            printf("%d ", up+tmp+1+i);
        }
        printf("\n");
        for(int i=0; i<tmp; i++){
            printf("%d ", left+i);
        }
        printf("\n");
        printf("cnt=%d\n", cnt++);
    }
}

int main(){
    cin >> n;
    miner(n, 1, n, 1, n);
    return 0;
}

然后不出意外会发现：这么做消耗的次数比一行一行遍历还要高——因为每一个禁区边缘的区域都转化成了一个小方格去开采。印象中这样的做法大概进行了8700多次……

我们需要一个更好的方法

这不一定是更好的方法（因为重复挖了很多方格），但是是最节省集合选择次数（上限50次）的方法。

主要的思路是——同时搜索左上、右上、左下、右下，以节省次数，这样就

因为在使用分治递归调用的过程中，无法使得每一轮的$X$和$Y$对应被打印，也无法使得不干扰的区域之间同时被搜索，因此使用了vector<vector<int>> solution(50, vector<int>())将策略存储在容器之中；因为对答案有需求，所以我们在进行遍历的时候也要从小次序开始，使用push_back直接把元素放入末尾即可。

对于$n$比较小的情况，比如$n \leq 7$的情况，仍然是一行一行遍历的情况次数更少；因此并没有进行操作，直接暴力push_back(._.)

对于$n$比较大的情况我们就不能遍历了。

我还是使用了分奇偶的思路，并同时搜索四块区域。

先以偶数$n=10$为例：

跟明显：左上和右下是$n = 4$的情况，右上和左下是一个完整的正方形，中间有两行十字形的区域。我的处理策略是：递归处理左上和右下的子情况，耗时：miner(4)；左下和右上完整的正方形：2次（整个挖去正方形）；十字形直接遍历横竖：2次；

那么如果每轮进行4次的话，大致会进行46次；

至于奇数，以$n=11$为例：不多解释啦，看图就能明白啦

#include "iostream"
#include <cstdio>
#include <vector>

using namespace std;

int n;
vector<vector<int>> solutionX(50, vector<int>());
vector<vector<int>> solutionY(50, vector<int>());
int check;


void mineSq(int left, int right, int up, int down){
    for(int i=up; i<= down; i++){
        printf("%d ", i);
    }
    printf("\n");
    for(int i=left; i<= right; i++){
        printf("%d ", i);
    }
    printf("\n");
}


void miner(int s, int left, int right, int up, int down, int opr){
    if(s == 3){
        solutionX[opr].push_back(up);
        solutionY[opr].push_back(left+1);
        solutionY[opr].push_back(right);
        opr++;
        solutionX[opr].push_back(up+1);
        solutionY[opr].push_back(right);
        opr++;
        solutionX[opr].push_back(up+2);
        solutionY[opr].push_back(left);
        opr++;
        check = opr;
        return;
    }
    if(s == 4){
        solutionX[opr].push_back(up);
        solutionY[opr].push_back(left+1);
        solutionY[opr].push_back(left+2);
        solutionY[opr].push_back(left+3);
        opr++;
        solutionX[opr].push_back(up+1);
        solutionY[opr].push_back(left+2);
        solutionY[opr].push_back(left+3);
        opr++;
        solutionX[opr].push_back(up+2);
        solutionY[opr].push_back(left);
        solutionY[opr].push_back(left+3);
        opr++;
        solutionX[opr].push_back(up+3);
        solutionY[opr].push_back(left);
        solutionY[opr].push_back(left+1);
        opr++;
        check = opr;
        return;
    }
    if(s == 5){
        solutionX[opr].push_back(up);
        solutionY[opr].push_back(left+1);
        solutionY[opr].push_back(left+2);
        solutionY[opr].push_back(left+3);
        solutionY[opr].push_back(left+4);
        opr++;
        solutionX[opr].push_back(up+1);
        solutionY[opr].push_back(left+2);
        solutionY[opr].push_back(left+3);
        solutionY[opr].push_back(left+4);
        opr++;
        solutionX[opr].push_back(up+2);
        solutionY[opr].push_back(left);
        solutionY[opr].push_back(left+3);
        solutionY[opr].push_back(left+4);
        opr++;
        solutionX[opr].push_back(up+3);
        solutionY[opr].push_back(left);
        solutionY[opr].push_back(left+1);
        solutionY[opr].push_back(left+4);
        opr++;
        solutionX[opr].push_back(up+4);
        solutionY[opr].push_back(left);
        solutionY[opr].push_back(left+1);
        solutionY[opr].push_back(left+2);
        opr++;
        check = opr;
        return;
    }
    if(s == 6){
        solutionX[opr].push_back(up);
        solutionY[opr].push_back(left+1);
        solutionY[opr].push_back(left+2);
        solutionY[opr].push_back(left+3);
        solutionY[opr].push_back(left+4);
        solutionY[opr].push_back(left+5);
        opr++;
        solutionX[opr].push_back(up+1);
        solutionY[opr].push_back(left+2);
        solutionY[opr].push_back(left+3);
        solutionY[opr].push_back(left+4);
        solutionY[opr].push_back(left+5);
        opr++;
        solutionX[opr].push_back(up+2);
        solutionY[opr].push_back(left);
        solutionY[opr].push_back(left+3);
        solutionY[opr].push_back(left+4);
        solutionY[opr].push_back(left+5);
        opr++;
        solutionX[opr].push_back(up+3);
        solutionY[opr].push_back(left);
        solutionY[opr].push_back(left+1);
        solutionY[opr].push_back(left+4);
        solutionY[opr].push_back(left+5);
        opr++;
        solutionX[opr].push_back(up+4);
        solutionY[opr].push_back(left);
        solutionY[opr].push_back(left+1);
        solutionY[opr].push_back(left+2);
        solutionY[opr].push_back(left+5);
        opr++;
        solutionX[opr].push_back(up+5);
        solutionY[opr].push_back(left);
        solutionY[opr].push_back(left+1);
        solutionY[opr].push_back(left+2);
        solutionY[opr].push_back(left+3);
        opr++;
        check = opr;
        return;
    }
    if(s%2 == 1){
        int mid = (s+1)/2;
        for(int i=left+mid; i<=right; i++){
            solutionY[opr].push_back(i);
        }
        for(int i=up; i<=up+mid-2; i++){
            solutionX[opr].push_back(i);
        }
        opr++;
        for(int i=left; i<=left+mid-2; i++){
            solutionY[opr].push_back(i);
        }
        for(int i=up+mid; i<=down; i++){
            solutionX[opr].push_back(i);
        }
        opr++;
        solutionX[opr].push_back(up+mid-1);
        for(int i=left; i<=right; i++){
            if(i==left+mid-2 || i==left+mid-1) continue;
            solutionY[opr].push_back(i);
        }
        opr++;
        solutionY[opr].push_back(left+mid-1);
        for(int i=up; i<=down; i++){
            if(i==up+mid-1 || i==up+mid) continue;
            solutionX[opr].push_back(i);
        }
        opr++;
        miner(mid-1, left, left+mid-2, up, up+mid-2, opr);
        miner(mid-1, left+mid, right, up+mid, down, opr);
        return;
    }
    if(s%2 == 0){
        int mid = s/2;
        for(int i=left+mid-1; i<=right; i++){
            solutionY[opr].push_back(i);
        }
        for(int i=up; i<=up+mid-2; i++){
            solutionX[opr].push_back(i);
        }
        opr++;
        for(int i=left; i<=left+mid; i++){
            solutionY[opr].push_back(i);
        }
        for(int i=up+mid+2; i<=down; i++){
            solutionX[opr].push_back(i);
        }
        opr++;
        solutionX[opr].push_back(up+mid-1);
        for(int i=left; i<=right; i++){
            if(i==left+mid-2 || i==left+mid-1) continue;
            solutionY[opr].push_back(i);
        }
        opr++;
        solutionX[opr].push_back(up+mid);
        for(int i=left; i<=right; i++){
            if(i==left+mid || i==left+mid-1) continue;
            solutionY[opr].push_back(i);
        }
        opr++;
        solutionX[opr].push_back(up+mid+1);
        for(int i=left; i<=left+mid-1; i++){
            solutionY[opr].push_back(i);
        }
        opr++;
        miner(mid-1, left, left+mid-2, up, up+mid-2, opr);
        miner(mid-1, left+mid+1, right, up+mid+1, down, opr);
        return;
    }
}


int main(){
    cin >> n;
    miner(n, 1, n, 1, n, 0);
    //cout << check << endl;
    for(int i=0; i<check; i++){
        for(int j=0; j<solutionX[i].size(); j++){
            printf("%d ", solutionX[i][j]);
        }
        printf("\n");
        for(int k=0; k<solutionY[i].size(); k++){
            printf("%d ", solutionY[i][k]);
        }
        printf("\n");
    }
    return 0;
}
